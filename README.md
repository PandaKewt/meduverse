# Meduverse
## Các bước cài đặt
Cài [melonloader](https://melonwiki.xyz/) vào trong game ở thư mục Game

Compile phần mềm và copy file EduverseMod.dll vào Mod và file UrGUI.dll vào UserLibs

Bấm RShift để mở menu
